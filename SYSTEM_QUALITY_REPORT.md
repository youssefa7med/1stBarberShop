### 1️⃣ **خطأ منطقي في `useQueueStatus.ts` - حساب النسبة المئوية** ❌

**الموقع:** `src/db/hooks/useQueueStatus.ts` - السطر 111-120

**المشكلة:**
```typescript
// ❌ معقد جداً وخطأ العملية المنطقية
const percentageWaited =
  currentBooking && (currentBooking.duration || 30) > 0
    ? Math.min(
        100,
        ((((currentBooking.duration || 30) * 60000 -
          (new Date(currentBooking.bookingTime).getTime() +
            (currentBooking.duration || 30) * 60000 -
            currentTime.getTime())) /
          ((currentBooking.duration || 30) * 60000)) *
          100)
      )
    : 0
```

**التحليل:**
- الصيغة الحالية معقدة جداً وغير واضحة
- تحتوي على أقواس متداخلة قد تسبب أخطاء
- الحساب قد يعطي نتائج خاطئة

**الحل الصحيح:** ✅
```typescript
const percentageWaited = currentBooking && (currentBooking.duration || 30) > 0
  ? Math.min(
      100,
      Math.max(
        0,
        ((currentTime.getTime() - new Date(currentBooking.bookingTime).getTime()) /
          ((currentBooking.duration || 30) * 60000)) * 100
      )
    )
  : 0
```
